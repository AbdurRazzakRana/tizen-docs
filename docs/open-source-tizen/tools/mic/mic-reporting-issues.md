# Reporting issues

### Reporting issues

Report bugs or feature requests at JIRA: [https://bugs.tizen.org](https://bugs.tizen.org/)

Detailed steps:

- Click "create issue"
- Select Projects: "Development Tools"
- Select Components: "MIC"
